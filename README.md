Aptitude Arena is a web-based aptitude practice platform designed to help students improve their problem-solving skills through category-wise questions, timed practice, score tracking, and performance analysis.

It is built as an interactive learning platform for students preparing for placements, competitive exams, and aptitude assessments.

---

## 🚀 Features

- 🔐 User Registration & Login Authentication
- 👤 Secure User Validation
- 📚 Category-wise Aptitude Practice
- ⏱️ Timed Quiz Experience
- 📊 Instant Results & Score Display
- 🏆 Rank / Performance Tracking
- 📈 User Dashboard
- 💾 Database Storage for Users & Scores
- 🎯 Beginner Friendly Interface
